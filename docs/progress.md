### 2026-02-26 - 创作提示词管理功能

**做了什么：**
1. 添加创作提示词管理功能，支持作者定义写作风格和要求
2. 支持创建多套提示词（写作风格模板），每套包含：
   - 结构化字段：语气风格、人称视角、目标读者、内容规则列表
   - 自由定义区：作者可以自由书写任何写作要求
3. 设置页面使用折叠面板设计，节省空间
4. 编辑器中添加提示词选择徽章，可快速切换提示词
5. AI 辅助功能（润色、聊天）自动应用选中的提示词
6. 支持临时调整（只对当前文章生效）
7. 支持设置默认提示词（新建文章时自动使用）

**修改了哪些文件：**
- `src/renderer/App.tsx` - 添加 `CreationConstitution` 类型，更新 `Topic` 和 `Config` 类型
- `src/renderer/pages/Settings.tsx` - 重写为折叠面板 UI，添加提示词管理界面
- `src/renderer/pages/Editor.tsx` - 添加提示词选择徽章和侧边面板，AI 调用时注入提示词
- `CLAUDE.md` - 更新项目信息和功能说明

**需要注意的问题：**
- 提示词数据保存在 `config.json` 的 `creationConstitutions` 字段
- 每篇文章的提示词选择保存在 `Topic.constitutionId` 字段
- 临时调整保存在 `Topic.temporaryAdjustments` 字段
- 所有 AI 调用（润色、聊天）都会自动注入提示词作为 system prompt
- UI 文字统一使用"创作提示词"而非"创作宪法"
